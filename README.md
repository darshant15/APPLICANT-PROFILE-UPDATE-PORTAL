# KHB Applicant Profile Update Portal

## Demo accounts

- Applicant: `APP202600123` / `Applicant@123`
- Admin: `REG-014` / `Admin@123`
- OTP for both: `123456`

## Applicant flow

Login → OTP → Applicant Dashboard → choose Address, Phone, Email or Name update → submit request → track status.

## Admin flow

Login → OTP → Admin Dashboard → review all update request types → View, Approve or Reject.

## Run

```bash
dotnet clean
dotnet restore
dotnet run
```

Open the localhost URL printed in the terminal.
